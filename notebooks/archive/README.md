# Notebook Archive

Notebooks in this folder document historical validation runs or redundant flows.
They remain available for reference but are no longer part of the recommended
entry path.

- `preliminary_smoke_test.ipynb` – original IBM hardware smoke test walkthrough.
- `review_smoke_test_results.ipynb` – manual analysis of the October 2025 smoke
  test outputs.
- `s_t01_ghz_classical_shadows.ipynb` – early version of the GHZ baseline demo
  superseded by `quickstart_shot_persistence.ipynb` and the comprehensive test
  suite.

Use the curated notebooks in the repository root for up-to-date tutorials.
