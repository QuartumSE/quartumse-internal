Quantum Approximate Optimization for MAX-CUT on 5-node ring

Diagram generation failed

Description

QAOA for MAX-CUT on 5-node ring graph. Optimal cut is 4 edges (alternating partition). Encodes superposition over all 32 partitions.

|psi(g,b)> = e^{-ib Sum Xi} e^{-ig Sum ZiZj} |+>^n Cost: H_C = Sum_{edges} (1 - ZiZj)/2
Applications

Combinatorial optimization, scheduling, logistics, hybrid classical-quantum algorithms

Why This Circuit Matters for Benchmarking

QAOA observables (ZiZj) all commute, so grouped measurement is efficient. Tests if shadows compete.
